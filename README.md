# TM_Test

Test package to implement a transfer matrix based 1d simulation of a MADMAX-like booster setup.

## Installation
Run:
```julia
using Pkg; Pkg.add(url="https://github.com/bergermann/TM_Test.jl.git")
```

## Usage
See documentations of ```transfer_matrix```, ```pos2dist```, ```dist2pos``` and ```gen_freqs```.
